Art_1: ¿Que es HTML?

HTML significa lenguaje de marcado de hipertexto. Se utiliza para diseñar páginas web utilizando un lenguaje de marcado. HTML es una combinación de hipertexto y lenguaje de marcado. El hipertexto define el vínculo entre páginas web. Se utiliza un lenguaje de marcado para definir el documento de texto dentro de la etiqueta que define la estructura de las páginas web.
